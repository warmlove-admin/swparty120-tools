## 收工交班固定項目

1. 每次收工前需交代本機 FastAPI 是否仍在執行，包含本機網址、PID 與 `/health` 檢查結果。
2. 每次收工前需交代 ngrok/localtunnel 狀態：目前 public URL、LINE webhook URL（public URL + `/line/webhook`）、是否仍可連線。
3. 若 public URL 有變更或 tunnel 已停止，交班要明確寫「需要到 LINE Developers 後台更新 Webhook URL」或「下次開工需重開 tunnel 後再更新」。
4. 若 ngrok 因 authtoken 或其他錯誤無法使用，交班要寫出錯誤原因，並註明可暫用 localtunnel 測試。
5. 交班時提醒下一位 agent：不要清除資料庫、LINE 來源對應、電訪、申訴資料，除非使用者明確要求。

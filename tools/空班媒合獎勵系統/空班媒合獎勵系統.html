📖 獎勵規則說明

時段倍數：06:00–09:30 ×1.5｜09:30–11:00 ×1.25｜11:00–13:00 ×1.0｜13:00–15:00 ×1.5｜15:00–16:30 ×1.25｜16:30–18:30 ×1.0

不計獎勵：06:00 前 / 18:30 後的時段不納入加權計算

跨時段加權：依實際落在各區間的時數比例加權，例如 08:30–10:30 = 50%×1.5 + 50%×1.25 = ×1.375

時數級距：S（未滿2h）= 150元｜A（2–4h）= 300元｜B（4–6h）= 500元｜C（6h以上）= 700元

期間比例：未滿2週=0%（不納入）｜短期2–4週=30%｜長期或≥4週=100%｜短期超5週補發至100%

AA加碼：AA05 +200元 / AA06 +200元，獨立勾選可同時申報（+400元）

居督挪移協調（有紀錄）：與個管相同金額

居督僅確認班表：象徵性 S=15 / A=30 / B=50 / C=70 元（同樣套用時段加權）

收入效益基準：政府給付 350元/h｜居服員薪資 230元/h｜毛利 120元/h｜AA05/AA06 各 200元/次
